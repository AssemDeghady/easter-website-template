Personalize it

I'll explain how to change the logo, the color schemes, animations, ...

Navbar

To add new or change links that goes to other sections in the page make sure href attribute has the same string as the section id

Example:

  <a href="#portfolio">Portfolio</a>
  <section id="portfolio"></section>

If you want to remove the underline animation for the nav links remove this tag from list item

  <div class="mask"></div>

Page jQuery Loader

If you dont like loaders and want to remove it from the page remove this code block from index.html

  
  <script type="text/javascript" src="js/queryloader2.min.js"></script>
  <script>
    window.addEventListener('DOMContentLoaded', function() {
      "use strict";
      new QueryLoader2(document.querySelector("body"), {
        barColor: "#9BCA3E",
        backgroundColor: "#111",
        percentage: true,
        barHeight: 1,
        minimumTime: 200,
        fadeOutTime: 1000
      });
    });
  </script>

Selecting Color Scheme

After Deciding a color scheme there are a few things to do:

- Remove the "alternate" rel from the stylesheet link if exists

- Remove everything enclosed by a comment with title "styleswitcher" inside the index.html and main.js files

- If you are keeping the page loader you will need to change the bar color only if u have chosen another color other than the green color

at the end of the head inside index.html

  window.addEventListener('DOMContentLoaded', function() {
    "use strict";
    new QueryLoader2(document.querySelector("body"), {
      barColor: "#9BCA3E", // Change this to the color inside the color.css in color attribute of class .highlight
      backgroundColor: "#111", // You Can also Change this if needed
      .......
    });
  });

Animations

If you don't like wow animations you can remove "wow" class and any class near it from the index.html file

  // For Example
  <div class="section-title wow fadeInUp">

Or you can just remove the wow js code in the main.js file

  //  ============== Wow Plugin =============== //
  new WOW().init();
  //  ============== Wow Plugin: END =============== //

Create Your Own Color

If you don't like any of the colors and want to make your own color, its very easy.

go to green.css file and look for these color codes

"#9BCA3E"
"rgba(158, 219, 54,"

search and find all lines containing "#9BCA3E" and change it to the hex code u want

do the same for the lines containing "rgba(158, 219, 54," but change it to the same color u chose but in rgb form

You can rename the file but make sure its the only skin file inside index.html

Changing the CSS

If you'd like to overwrite some css rules or add some yourself, you can do this in the "style.css" file located in the "css" folder.

If you want to add new css rules create a custom.css file or use any other name and add it after the skin css file in the index.html head tag

Editing Map

First You need to add your own API Key from google developer console Get A Key, Then Replace the template key here in index.html:

  <script src="http://maps.google.com/maps/api/js?key=AIzaSyAHlg64m9bm75-waAZl2xOPTlbWV0oc5ps"></script>

You edit the map marker location inside the main.js file here:

  //  ============== Contact Us =============== //
  var map = new GMaps({
    el: '#map',
    lat: 40.730610,
    lng:  -73.935242,
    zoom: 13      
  });

  map.addMarker({
    lat: 40.730610,
    lng:  -73.935242,
    click: function(e) {
      map.setCenter(e.position.lat(), e.position.lng());
      map.setZoom(13);
    },      
    infoWindow: {
      content: '>div class="marker-info-window"<>h4<Contact Personnel>/h4<>p<Location Address>/p<>/div<'
    }
  });
  // ============== Contact Us: END =============== //

Also you can duplicate the map.addMarker({}) block to add more markers.
